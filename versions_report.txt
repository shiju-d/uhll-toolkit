UHLL Version Report
*******************

This report details all UHLL updates and additions on a version by version basis. The sections are sorted by UHLL specification
version, with each section detailing all additions and changes to each of the following:

o Version Information
o New And Updated Messages
o New And Updated Message DFIDs
o New And Updated Message Notes
o New And Updated Categories
o New And Updated Category List Values

A new version of this report will be published, in tandem, with all future revisions of UHLL. Please remember
to visit the HPD website (https://my.comtrol.com) to download the latest version of this report.
